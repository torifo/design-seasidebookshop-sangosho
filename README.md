# 珊瑚礁の頁 — design-seasidebookshop-sangosho

> **架空ブランド注記**: 珊瑚礁の頁は本デザイン研究のために作成した架空書店です。実在の店舗・商品ではありません。

## Overview

**seaside bookshop design research series** の「南西諸島の珊瑚礁」ペルソナ作品。

旅人と島の子どもが混ざる小さな店として、絵本、民話、海図、自然観察、紙ものを明るく配置する。

## Design Approach

- ヒーロー写真を有機的な島形に切り抜き、手描き地図のようなピンを置く
- 棚は絵本、民話、海図、手紙に分け、子どもと旅人が同じ場所で読めるようにする
- プログラム情報を読み聞かせ、観察帖、手紙の時間として表示
- 小さな市の情報を置き、観光土産ではなく読書の周辺物として扱う
